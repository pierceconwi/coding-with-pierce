Coding with Pierce

As an aspiring web developer, selecting the best computer programming languages to learn is a primary component of a good career plan. However, with literally dozens upon dozens of computer programming languages currently in existence, narrowing down which languages to learn (and in what order) can be a daunting task, especially to those unfamiliar with some of the programming world's more opaque terminology.

This small website is intended to offer some quick and dirty insights into the languages that make the most sense to consider learning first (at this specific moment in time, to one specific person [i.e. me]). Ideally, in the future it will save someone else the trouble of digging this information up for themselves.
